**My Explorer** is a Mac-focused file manager with two panels side by side (or one above the other). It is based on the open-source **Double Commander** project and keeps Double Commander's useful dual-pane comparison workflow.

This fork preserves attribution to Double Commander and remains subject to the open-source license terms included in this repository.

## Attribution

## Where to start

### Download

Download My Explorer from this repository's GitHub releases.

### Develop

The source code still contains upstream Double Commander internals where keeping those names avoids unnecessary build-system churn. Product-facing names, macOS packaging, and documentation are branded as My Explorer.
